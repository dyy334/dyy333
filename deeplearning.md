5.1 理论计算题

已知线性回归模型：y\_pred\_i = w \* x\_i + b  
均方误差损失：L = (1/n) \* sum\_{i=1 to n} (y\_i - (w \* x\_i + b))^2

令 d\_i = y\_i - (w \* x\_i + b)，则 L = (1/n) \* sum(d\_i^2)

1.  对 w 求偏导：  
    ∂L/∂w = (1/n) \* sum( 2 \* d\_i \* ∂d\_i/∂w )  
    由于 ∂d\_i/∂w = -x\_i  
    所以 ∂L/∂w = (1/n) \* sum( 2 \* (y\_i - w x\_i - b) \* (-x\_i) )  
    \= (1/n) \* sum( -2 x\_i (y\_i - w x\_i - b) )  
    \= -(2/n) \* sum( x\_i (y\_i - w x\_i - b) )  
    等价于 ∂L/∂w = (2/n) \* sum( -x\_i (y\_i - w x\_i - b) ) 或更常见写成：  
    ∂L/∂w = (2/n) \* sum( x\_i (w x\_i + b - y\_i) )
2.  对 b 求偏导：  
    ∂L/∂b = (1/n) \* sum( 2 \* d\_i \* ∂d\_i/∂b )  
    由于 ∂d\_i/∂b = -1  
    所以 ∂L/∂b = (1/n) \* sum( 2 (y\_i - w x\_i - b) \* (-1) )  
    \= (1/n) \* sum( -2 (y\_i - w x\_i - b) )  
    \= -(2/n) \* sum( y\_i - w x\_i - b )  
    \= (2/n) \* sum( w x\_i + b - y\_i )

最终表达式：  
∂L/∂w = (2/n) \* sum\_{i=1}^n \[ x\_i (w x\_i + b - y\_i) \]  
∂L/∂b = (2/n) \* sum\_{i=1}^n ( w x\_i + b - y\_i )

已知：x1, x2, ..., xn 独立同分布于正态分布 N(µ, σ²)

1.  似然函数 L(µ, σ²)

正态分布的概率密度函数：  
f(xi | µ, σ²) = (1 / sqrt(2πσ²)) \* exp\[ -(xi - µ)² / (2σ²) \]

由于独立，似然函数为各密度乘积：  
L(µ, σ²) = ∏\_{i=1}^n (1 / sqrt(2πσ²)) \* exp\[ -(xi - µ)² / (2σ²) \]

整理：  
L(µ, σ²) = (2πσ²)^(-n/2) \* exp\[ - (1/(2σ²)) \* ∑\_{i=1}^n (xi - µ)² \]

2.  证明 µ 的 MLE 为 µ̂ = (1/n) ∑\_{i=1}^n xi

取对数似然（方便求导）：  
ℓ(µ, σ²) = ln L = -(n/2) ln(2πσ²) - (1/(2σ²)) ∑\_{i=1}^n (xi - µ)²

对 µ 求偏导：  
∂ℓ/∂µ = - (1/(2σ²)) \* ∑\_{i=1}^n 2(xi - µ) \* (-1) \[注意对 µ 求导，内部 (xi - µ)² 导数为 -2(xi - µ)\]  
实际上更准确：  
∂/∂µ ∑ (xi - µ)² = ∑ 2(xi - µ) \* (-1) = -2 ∑ (xi - µ)

所以：  
∂ℓ/∂µ = - (1/(2σ²)) \* \[ -2 ∑ (xi - µ) \] = (1/σ²) ∑ (xi - µ)

令 ∂ℓ/∂µ = 0：  
∑ (xi - µ̂) = 0  
∑ xi - n µ̂ = 0  
µ̂ = (1/n) ∑ xi

得证。

3.  证明 σ² 的 MLE 为 σ̂² = (1/n) ∑\_{i=1}^n (xi - µ̂)²

对 σ² 求偏导。令 θ = σ²，则：  
ℓ = -(n/2) ln(2πθ) - (1/(2θ)) ∑ (xi - µ)²

∂ℓ/∂θ = -(n/2) \* (1/θ) + (1/(2θ²)) ∑ (xi - µ)²  
\= -n/(2θ) + (1/(2θ²)) ∑ (xi - µ)²

令 ∂ℓ/∂θ = 0，两边乘以 2θ²：  
\-n θ + ∑ (xi - µ)² = 0  
θ = (1/n) ∑ (xi - µ)²

注意这里 µ 未知，但在 MLE 中需代 µ̂。故：  
σ̂² = (1/n) ∑\_{i=1}^n (xi - µ̂)²

得证。
